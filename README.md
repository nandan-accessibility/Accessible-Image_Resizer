# Accessible Image Resizer 

Developed by Nandan Kumar Singh

Accessible Image Resizer is a client-side web-based tool that allows users to resize and compress images directly in the browser.

## Features

- Upload and preview image
- Custom width and height input
- DPI selection (72, 300, Custom)
- Target file size compression (KB-based)
- Live file size preview
- Responsive design (Mobile, Tablet, Desktop)
- Accessibility-friendly structure
- Screen reader support

## Accessibility Highlights

- ARIA labels on form controls
- Status messages using role="status"
- Keyboard accessible buttons and inputs
- Screen reader friendly structure

## Technologies Used

- HTML5
- CSS3
- JavaScript
- HTML5 Canvas API
